# JS_for_Postman
JS for Postman
1. Створити експенс для авто. Параметри експенсу задати в пререквесті першого запиту у вигляді об'єкту. 
2. Перевірити, що параметри експенсу застосовані вірно.
3. Змінити параметри експенсу. Параметри для зміни експенсу задати об'єктом у пре-реквесті першого запиту. 
4. Перевірити, що параметри експенсу застосовані коректно.
5. Перевірити параметри авто з урахуванням експенсу.
